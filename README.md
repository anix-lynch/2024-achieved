## üöÄ Making Repositories Private

To make all these repositories private, use the following GitHub CLI commands:

```bash
# Install GitHub CLI if you haven't already
# brew install gh

# Make all 2024 repos private (bulk operation)
gh repo list anix-lynch --limit 200 --json name | jq -r '.[].name' | while read repo; do
  gh repo edit anix-lynch/$repo --visibility private
done
```

Or, to be more selective, run individual commands:

```bash
# For specific repos (example)
gh repo edit anix-lynch/mnist-digit-recognition-pytorch --visibility private
gh repo edit anix-lynch/Income_Classification --visibility private
gh repo edit anix-lynch/Titanic --visibility private
# ... repeat for each repo
```
